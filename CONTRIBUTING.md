# Contributing

Thanks for helping improve RedWeaver.

## Getting started

1. Fork the repository and clone your fork.
2. Copy `.env.example` to `.env` and add at least one LLM API key (or use Ollama locally).
3. Run `docker compose up --build` or follow the local dev commands in [README.md](README.md).

## Pull requests

- Keep changes focused on a single concern when possible.
- Match existing code style (Python: `ruff`/`black` if configured; TypeScript: project `tsconfig`).

## Security

See [SECURITY.md](SECURITY.md) for vulnerability reporting and deployment notes.
